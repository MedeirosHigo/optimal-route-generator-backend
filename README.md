Kruskal Route Planner - Backend
Ce projet est un service backend développé en Java avec Spring Boot, qui utilise l’algorithme de Kruskal pour optimiser les itinéraires de livraison sur la base de graphes pondérés.

🧠 Objectif
L’objectif de ce projet individuel est d’appliquer, de manière intégrée, les connaissances acquises dans plusieurs disciplines telles que la Recherche Opérationnelle, les Algorithmes, l’Interface Homme-Machine (IHM) et le Développement Backend.
Le projet repose sur les concepts de graphes et l’étude de la Recherche Opérationnelle, en particulier l’algorithme de Kruskal, afin de résoudre le problème d’optimisation des trajets.
L’Api fournit un itinéraire optimisé (Arbre de Couverture Minimal) à partir d’un ensemble de points de livraison, et peut être intégrée à un frontend pour la visualisation des résultats.

⚙️ Technologies utilisées
Java 17+
Spring Boot
Maven

Lombok
API REST
Architecture MVC
