# Kruskal Route Planner - Backend

Ce projet est un service backend développé en **Java avec Spring Boot**, qui utilise l’**algorithme de Kruskal** pour optimiser les itinéraires de livraison à partir de graphes pondérés.

## 🧠 Objectif

L’objectif de ce projet individuel est d’appliquer de manière intégrée les connaissances acquises dans plusieurs disciplines telles que la **Recherche Opérationnelle**, les **Algorithmes**, l’**Interface Homme-Machine (IHM)** et le **Développement Backend**.  
Le projet repose sur les concepts de **graphes** et l’étude de la **Recherche Opérationnelle**, plus précisément l’**algorithme de Kruskal**, pour résoudre le problème d’optimisation des trajets. L’application fournit un itinéraire optimisé (arbre couvrant minimal) à partir d’un ensemble de points de livraison, avec la possibilité d’intégration dans un frontend pour visualisation des résultats.

## ⚙️ Technologies utilisées

- Java 17+  
- Spring Boot  
- Maven  
- OpenAPI (Swagger)  
- API REST  
- Architecture MVC  
- DTOs (Request/Response)
